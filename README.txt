# GornitsaTestTask
The program provides some information about a product by product article.

Пользователь вводит артикул стеклопакета, который может быть однокамерный (стекло/рамка/стекло), либо двухкамерный (стекло/рамка/стекло/рамка/стекло).
В свою очередь, программа выводит информацию о стеклопакете:
- количество камер;
- общая толщина;
- толщина всех стекол.

Примеры артикулов:
4 TOP-N/14/6_м1 (однокамерный, общая толщина - 24, толщина стекол - 10);
4 TOP-N/14ar/4 м1/14ar/4_StClBr (двухкамерный, общая толщина - 40, толщина стекол - 12).
